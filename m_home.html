  • How Does it Work?

    You simply send us instructions on how to perform the boring task (or a video if it’s hard to explain) and some files/pages for us to test the program on.

    Then, using robust scripts, automotive computer programs, and a breadth of programming knowledge, we compile a full, simple to use solution that (typically) doesn’t require any external programs.

    We utilize a plethora of Excel, Word, PDF, Web Scraping, Machine Learning, and Deep Learning libraries to automate tasks, no matter how specific/broad or creative/technical they are.

  • How Much Does it Cost?

    Pricing per project depends entirely on the difficulty of the task. A project can range from $5 for a simple script to $3,000 for more complex Machine Learning or Web Scraping programs.

    The full price of the task will be presented and agreed upon before we build the program. Once we’re finished building, we’ll show you how we automated your boring task. If it doesn’t accomplish what you want it to, you pay nothing.

    Once we send you the program, you’ll enjoy the benefits of being a boringAI client. Namely, if the task we automated undergoes a change, we will work on revising your automation program for an additional fee based on a small percentage of the original task’s price.

    Lastly, if you refer a friend that purchases a task, you will receive a 25% rebate after their purchase. Contact us for a free task cost-appraisal.
